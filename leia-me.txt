Configurar Database:

1. Crie um banco de Dados no seu MYSQL-SERVER
2. Na raiz do projeto abra o arquivo '.env'
3. Altere a linha "DB_DATABASE=laravel" para "DB_DATABASE=nome_do_database_criado"
4. Altere a linha "DB_USERNAME=root" para "DB_DATABASE=nome_do_usu√°rio_do_database"
5. Altere a linha "DB_PASSWORD=" para "DB_DATABASE=senha_do_database"

Criando as tabelas do banco de dados:

1. Abra um terminal do seu sistema operacional
2. Digite o comando "cd caminho_da_pasta_raiz_do_seu_projeto"
3. Digite o comando "php artisan migrate"
